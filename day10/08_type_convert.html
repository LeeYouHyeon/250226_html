타입 변환 casting (형변환)

  • 문자 -> 숫자 : Number("string"), parseInt("1"), parseFloat('0.5')
  • 숫자 -> 문자 : 123.toString()
  • 문자 -> boolean : Boolean("true")
  • boolean -> 문자 : String(true)
  • number -> boolean : Boolean(1)
    0, NaN, undefined, null은 false, 나머지는 true
  • boolean -> number : Number(true)
