String : 가장 많이 다루는 데이터 타입

  • java에서의 String method와 거의 유사함
  • 문자 데이터는 index(0부터 시작)를 가지고 있음
  • 문자열 값 내에 br 태그로 개행을 하는 것은 추천하지 않음
  • length : 문자열의 총 길이
  • new String() : 객체 타입으로 선언 가능 / 거의 안 씀
  • 추출 관련 메서드 : substring(), substr(), slice()
  • 문자 대체 관련 메서드 : replace(str, old, new) : 처음 찾은 값만 가능
  • trim() : 양쪽 공백 제거
  • padStart(str, 길이, 채울 문자) / padEnd()
    예 : 010104-1******(padEnd())
  • charAt(idx) / indexOf(str)
  • startWith(str, 문자) / endWith()
    문자가 str의 맨 앞/뒤에 존재하는지 확인
    return true / false
    예 : 첨부파일 확장자 체크
