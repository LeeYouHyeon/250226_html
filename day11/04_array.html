JavaScript에서 배열은 유동 배열 : 크기, 데이터 변경 가능

  • const myArray = [1, 2, 3, 'a', 'b', {}, []];
  • 배열의 데이터 타입을 다르게 넣어도 가능
    그러나 처리가 복잡해지기 때문에 일반적이진 않음
    배열은 같은 타입으로 채우는 것이 일반적
  • const members = [{}, {}, {}, {}];
  • let myArray = [];처럼 빈 배열을 만든 후 데이터를 추가하는 게 일반적
  • index를 가짐 => 0부터 시작
  • length : 배열의 총 길이
  • const arr = new Array(1, 2, 3); 가능은 하나 잘 쓰지 않음
