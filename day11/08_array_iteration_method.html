Array Iteration Method : 순환 구조에서 사용하는 메서드들

  • arrray.reduce(callback function(total, value, index, self), [init value]);
    연산의 누적값을 리턴
  • array.every((value, index, self) => boolean)
    콜백 함수의 리턴 조건에 모든 배열의 값이 만족하면 true
  • array.some((v, i, s) => boolean)
    or에 해당
  • array.indexOf('str', [start index]);
    해당 문자열이 존재하는 index를 리턴
  • array.lastIndexOf('str', [start index]);
  • array.find((v, i, s) => boolean);
    조건에 맞는 첫번째 값을 리턴
  • array.findIndex((v, i, s) => boolean);
    조건에 맞는 첫 값의 index를 리턴
  • array.form('객체') : 순서가 있거나 순환구조를 가지고 있는 객체를 배열로 변환
  • array.keys() : 배열에서 사용중인 index를 새로운 배열로 리턴
    주로 object에서 사용
  • array.entries() : index: value의 구조로 새로운 배열을 만들어서 리턴
  • array.includes('str') : 배열에 str이 있는지 체크하여 boolean으로 리턴
