배열 관련 메서드 : 유사 배열에서는 사용이 제한될 수 있음
const myArray = [1, 2, 3, 'a', 'b', {}, []]; : 유사 배열(혼합 배열)

  • array.push(value) : 배열의 끝에 value 추가
    return : 배열의 길이
  • array.pop() : 배열의 마지막 요소를 제거
    return : 제거한 요소
  • array.shift() : 배열의 첫 요소를 제거(pop과 반대)
    return : 제거한 요소
  • array.unshift(value) : 배열의 첫 요소에 value를 추가
    return : 배열의 길이
  • array.join(value) : 배열의 값들 사이사이에 value를 추가해서 하나의 문자열로 리턴
