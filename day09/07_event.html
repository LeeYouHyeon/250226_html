html 객체에 줄 수 있는 전역객체로써, 모든 태그 객체가 참조할 수 있다.

  • a tag / button tag에 이벤트를 주소 참조시켜 사용 클릭에 대한 반응을 기대할 수 있게 만드는 것이 일반적
  • a tag는 href 이동속성을 막는 코드가 필요
  • 객체에 이벤트를 부여하는 방법은 크게 3가지로 구분
  • 각자 목적과 작동방식이 다름
  • 화면의 출력영역을 document 영역(객체)라고 한다.
  • DOM 영역
  • tag = html element, 속성 = attribute, 일반 텍스트 = textNode라고 함
  • html로 출력하기 : innerHTML라는 property를 사용
  • text로 출력하기 : innerText라는 property를 사용
  • html 요소를 가져올 때 getElementById / getElementByClassName
  • html 요소를 가져올 때 querySelector('selector') => # .
