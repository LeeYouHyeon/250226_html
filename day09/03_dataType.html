변수나 상수에 할당할 수 있는 값의 종류

  • 숫자 : Number
    정수, 상수, NaN(not a number : 숫자가 아님)
  • 문자 : String
  • boolean : true / false
  • 객체 : Object => 1개 이상의 데이터가 특정한 구조를 가지고 있는 형태
    {키 : 값} {}
    아무것도 없는 것도 하나의 데이터로 인정해 객체로 구성 가능
  • 배열 : Array => [1, 2, 3, 4] 형태의 집합
    java에서의 list
    index, length가 있음
  • 알 수 없는 값 : null => 인지가 안 됨
  • 미지정 : undefined => 값이 할당되지 않음

  1. 기본 자료형 : 값이 메모리의 주소값을 가리키는 곳에 직접 할당되는 값
    number, boolean, string(참조 자료형이지만 기본 자료형처럼 사용)
  2. 참조 자료형 : 값이 저장되어있는 주소를 할당
    기본 자료형 이외 전부(string, array, object, function, null, undefined)
  3. == , != : 값만 비교
    123 == '123'
    ===, !== : 값과 타입을 같이 비교
    123 !== '123'
