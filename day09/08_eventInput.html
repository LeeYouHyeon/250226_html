input, textarea, select, html element 텍스트를 받아오기

  • input tag는 value 속성에 값이 매핑되어있어 value 속성으로 값을 가져올 수 있음
  • h1처럼 열고 닫는 태그가 있는 경우 innerText 속성으로 가져온다.
  • select => option / radio / checkbox
    value의 값을 태그 내 속성으로 임의로 부여해야 선택한 값을 가져올 수 있음
  • value / innerText 모두 가져오는 값은 string 취급
  • string 처리된 값을 숫자로 쓰기 위해서는 변환이 필요함
